Cheat Sheet to understand HTML Tags

HTML Tags Reference Table

Tag Name Description Other
<table> Table The wrapper element for all HTML tables. The element that goes round the entire table so we state this first
<thead> Table Head The set of rows defining the collumn headers within the table. This is the title of each of the columns
<tbody> Table Body The set of rows that contain actual data. Throughout the entire table
<tr> Table Row The table row container. as mentioned
<td> Table Data The table row container. as mentioned
<tfoot> Table Foot The set of rows that define the footer within the table. At the bottom, similar to the Thead
